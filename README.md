# Hotel Booking Analysis using RapidMiner & Power BI

## Project Overview
This project analyzes hotel booking data using Machine Learning and Business Intelligence techniques.

The project focuses on:
- Booking Cancellation Prediction
- Customer Review Prediction
- Guest Segmentation

## Tools & Technologies
- RapidMiner
- Power BI
- Machine Learning
- Data Visualization

## Models Used
### Model 1 – Cancellation Prediction
- Algorithm: Random Forest
- Accuracy: 75.46%

### Model 2 – Review Prediction
- Algorithms:
  - Decision Tree
  - Naive Bayes
- Best Accuracy: 86.52%

### Model 3 – Guest Segmentation
- Algorithm: K-Means Clustering
- Segments Identified: 4

## Key Features
- Data Preprocessing
- Feature Engineering
- Normalization
- Predictive Analytics
- Dashboard Visualization

## Business Impact
- Reduces booking cancellations
- Improves customer satisfaction
- Supports targeted marketing strategies
